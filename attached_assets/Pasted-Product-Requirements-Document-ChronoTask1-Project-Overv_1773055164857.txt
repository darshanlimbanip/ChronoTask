Product Requirements Document: ChronoTask1. Project OverviewChronoTask is a productivity-focused web application that combines traditional task management with integrated time tracking. Unlike a standard to-do list, it allows users to quantify their effort by logging hours directly against specific tasks.Objective: To provide a seamless MERN-stack interface for organizing daily work and analyzing time allocation.2. Target AudienceFreelancers: Who need to track billable hours for specific tasks.Students: Who want to see how much time they spend on different subjects.Remote Workers: Looking to improve their time management and focus.3. Functional Requirements3.1 Dashboard (The Nerve Center)The dashboard provides a high-level summary of the user's workload using "at-a-glance" statistics.Total Tasks: Counter for all active and completed tasks.Today’s Tasks: Filtered view for tasks due on the current date.Overdue Tasks: Highlighted counter for tasks where dueDate < currentData and status is not "Completed."3.2 Task ManagementUsers need a robust way to input and categorize their work.Task Creation: A form containing:Title: (Short description)Details: (Rich text or simple textarea)Due Date: (Date picker)Priority: Dropdown selecting from Urgent, High, Normal, Low.Status Transitions: Ability to move tasks from "Pending" to "In Progress" or "Completed."3.3 Timesheet & Time TrackerThe "Killer Feature" of the app.Active Tracker: A stopwatch interface that selects an existing task from a dropdown and starts a timer.Manual Entry: Option to manually add hours if the user forgot to start the timer.Automatic Sync: Once "Stop" is clicked, the duration is calculated and added to the totalHours field of the selected task in the database.4. Technical Stack (MERN)LayerTechnologyRoleFrontendReact.js & Tailwind CSSResponsive UI and State management.BackendNode.js & Express.jsAPI routing and business logic.DatabaseMongoDBStoring Task, User, and TimeLog documents.AuthJSON Web Tokens (JWT)Securing user data and private routes.5. User StoriesAs a user, I want to see my Overdue Tasks in red so I can address them immediately.As a user, I want to assign a Priority to a task so I know which one to track time on first.As a user, I want to click "Start Tracking" on a task so I don't have to manually guess how long I worked.6.
